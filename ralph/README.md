# Ralph Wiggum Setup (Project Local)

This repository is configured for Ralph Wiggum autonomous loops.

## Run

- Codex loop: `./scripts/ralph-loop-codex.sh`
- Codex loop with max iterations: `./scripts/ralph-loop-codex.sh 20`
- Planning mode: `./scripts/ralph-loop-codex.sh plan`
- Claude loop: `./scripts/ralph-loop.sh`
